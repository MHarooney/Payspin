# Payspin App UI Kit

## Overview
High-fidelity recreation of the Payspin iOS mobile app (428×926pt design canvas, rendered at 390px wide for the web).

## Screens
1. **Splash** — Gradient brand screen with logo
2. **Onboarding** — 3-step feature introduction  
3. **Sign Up / Login** — Auth forms with social login
4. **Phone Auth** — Phone number entry + OTP
5. **Home** — Dashboard with circle summary + banners
6. **Admin View** — Circle admin with contributors list
7. **Create Circle** — 3-step circle creation flow
8. **Notifications** — Empty state notification screen

## Components
- `StatusBar` — iOS status bar mock
- `TopBar` — Screen title bar with back arrow / notification icon
- `BottomNav` — 4-tab navigation bar
- `PrimaryButton` — Full-width pill CTA button
- `InputField` — Labeled text input
- `TabToggle` — Sign Up / Log In tab switcher
- `CircleCard` — Payment circle summary card
- `BannerCard` — Promotional/social banner
- `Stepper` — 3-step progress indicator
- `AvatarRow` — Contributor list item with avatar
- `ContributorList` — Scrollable contributor table

## Usage
Open `index.html` for the interactive click-through prototype.
